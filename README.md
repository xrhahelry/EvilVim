
# EvilVim

## Description
  My nvim config written in Lua. I am still constantly improving on this config. You are free to contribute to the project.

## Plugins Used

1. Themes
    - gruvbox
    - gruvbox-material
    - material
2. Lsp
    - completion-nvim
    - nvim-compe
    - nvim-lsp-installer
    - nvim-lspconfig
    - vim-vsnip
    - nvim-treesitter
3. Fuzzy Finding and Basic Utils
    - nvim-autopairs
    - nvim-comment
    - nvim-gps
    - nvim-tree
    - nvim-web-devicons
    - telescope.nvim
    - which-key.nvim
